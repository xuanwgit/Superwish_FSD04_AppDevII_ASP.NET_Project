Superwish Toy Store

An ASP.NET Core e-commerce application for toy shopping

About the Application

Superwish is a modern e-commerce platform built with ASP.NET Core, designed to provide a seamless shopping experience for toy enthusiasts. The application features a responsive design, secure authentication, and a comprehensive admin dashboard.

User Features

  • User registration and authentication
  • Product browsing by categories
  • Shopping cart functionality
  • Order management
  • Responsive design for all devices

Admin Features

  • Product management
  • Category management
  • Order processing
  • User management
  • Background customization

Technical Features

  • ASP.NET Core 6.0
  • Entity Framework Core
  • PostgreSQL Database
  • Razor Pages
  • Bootstrap 5
