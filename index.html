Superwish Toy Store

A modern ASP.NET Core e-commerce platform for toy shopping

About the Application

Superwish is a comprehensive e-commerce platform built with ASP.NET Core, designed to provide a seamless shopping experience for toy enthusiasts. The application features a modern, responsive design, secure authentication, and a powerful admin dashboard for managing products, categories, and orders.

User Features

  • Secure user registration and authentication
  • Intuitive product browsing by categories
  • Real-time shopping cart functionality
  • Comprehensive order management system
  • Responsive design for all devices
  • Contact form and FAQ section

Admin Features

  • Complete product management system
  • Dynamic category management
  • Efficient order processing
  • User management and roles
  • Customizable background settings
  • Real-time inventory tracking

Technical Features

  • ASP.NET Core 6.0
  • Entity Framework Core
  • PostgreSQL Database
  • Razor Pages architecture
  • Bootstrap 5 for responsive design
  • Secure authentication with ASP.NET Identity
